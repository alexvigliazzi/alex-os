# Architect — Dev Domain

Carregar e operar conforme o agent definition em `~/.claude/agents/dev/architect.md`.

## Instruções

1. Ler `~/.claude/agents/dev/architect.md` — carregar persona completa do Architect
2. Ler `~/.claude/context/session-state.md` — contexto de sessão
3. Ler `~/.claude/context/project.md` — stack e padrões vigentes
4. Analisar `$ARGUMENTS` sob a perspectiva de arquitetura e design técnico
5. Responder com: análise → decisão de design → ADR (se aplicável) → plano de implementação

## Quando usar

- Design de feature ou sistema
- Criação de ADR (Architecture Decision Record)
- Design de schema ou API
- Seleção de tech stack
